# Securing-the-Network-with-pfSense-firewall-Blocking-Legacy-Insecure-Protocols

This lab demonstrates how to harden a network using a pfSense firewall by implementing a default-deny approach and blocking legacy and insecure protocols such as Telnet, FTP, and SMB. The objective is to reduce the attack surface by preventing plaintext communication, limiting lateral movement, and mitigating common exploitation techniques used by attackers.
